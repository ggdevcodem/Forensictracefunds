# ForensicTraceFund.com - Complete Production Rebuild

Production-ready full-stack application for blockchain forensics and crypto recovery services.

## 📋 Project Overview

**ForensicTraceFund** is a comprehensive platform offering:
- Blockchain forensics and analysis
- Crypto asset recovery services
- Wallet decryption
- DeFi compliance investigations
- Threat intelligence
- Law enforcement support

## 🏗️ Architecture

### Tech Stack

**Frontend:**
- Next.js 14+ (React framework)
- TypeScript
- Tailwind CSS (styling)
- React Hook Form (form management)
- Zod (validation)

**Backend:**
- Node.js + Express.js
- TypeScript
- PostgreSQL (database)
- JWT authentication
- Bcrypt (password hashing)

**Deployment:**
- Docker containerization
- Railway.app (recommended)
- or Vercel (frontend) + Railway (backend)

## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/ggdevcodem/Forensictracefunds.git
cd Forensictracefunds

# Docker setup (recommended)
docker-compose up -d

# Or manual setup
cd backend && npm install && npm run dev
cd ../frontend && npm install && npm run dev
```
